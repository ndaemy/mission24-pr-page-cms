# Mission24 페이지

https://mission24.link 의 CMS

## Prerequisites

- [Node.js](https://nodejs.org/)
- [Yarn](https://classic.yarnpkg.com/)

## Installation

```shell
$ yarn install  # or yarn
```

## How to start development server

```shell
$ yarn dev
```

## How to start production server

```shell
$ yarn build
$ yarn start
```
